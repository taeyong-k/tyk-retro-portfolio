# 🕹️ TYK Retro Portfolio

> "꾸준함과 꼼꼼함을 무기로, 평범하지만 믿고 맡길 수 있는 개발자가 되기 위해 만들었습니다."

나만의 개발 히스토리를 담은 레트로 감성 포트폴리오 사이트입니다.  
CRT 효과, 타자기 애니메이션, 옛날 윈도우 창과 디스크/영수증 테마 등,  
개성 있는 UI 구성과 깔끔한 코드 설계를 함께 고민하며 만들었습니다.

## 📁 프로젝트 구성

- **인트로 페이지**  
  CRT TV 애니메이션과 레트로 타이핑 효과 구현  
- **About Me**  
  옛날 윈도우 창 스타일로 구성된 자기소개 섹션  
- **Career**  
  타자기에서 출력되는 영수증 형태, 종이 뭉치/뜯김/테이프 표현  
- **Skill**  
  디스켓 아이콘 기반 카테고리, 클릭 시 상세 내용 표시  
- **Project**  
  비디오 테이프 테마 기반 프로젝트 리스트 + 상세 보기 전환  
- **반응형 지원**  
  모바일에서도 최소한의 가독성 확보

---

## ✨ 주요 기능

- 🔤 레트로 타자기 애니메이션 효과
- 🖥 CRT TV 인트로 with 브라운관 전원 켜짐 느낌
- 🧾 영수증 디자인 구현 (타자기 + 종이 질감 + 테이프)
- 💾 디스켓/비디오 테이프 테마 기반 스킬/프로젝트 UI
- 🎨 커스터마이징 가능한 폰트 및 윈도우 창 효과
- ⚙️ 모듈화된 컴포넌트 구조 (React + Tailwind 기준)
